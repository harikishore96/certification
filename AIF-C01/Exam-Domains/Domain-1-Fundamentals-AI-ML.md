## Domain Summary

### Key Focus Areas
1. **Terminology**: AI, ML, DL, supervised, unsupervised, reinforcement, training, inference
2. **Use Cases**: When to use AI/ML, value propositions, appropriate applications
3. **ML Lifecycle**: Data → Training → Evaluation → Deployment → Monitoring
4. **Data Understanding**: Types, formats, quality, preparation

### Common Question Types
- Define and differentiate AI/ML concepts
- Identify appropriate use cases for AI/ML
- Match business problems to learning paradigms
- Understand data requirements for different ML tasks
- Recognize stages in ML lifecycle

### Study Strategy
1. Master terminology and definitions
2. Understand relationships (AI ⊃ ML ⊃ DL)
3. Practice identifying use cases
4. Learn data type characteristics
5. Understand ML workflow end-to-end

### AWS Services to Know
- **SageMaker**: Complete ML platform
- **Ground Truth**: Data labeling
- **Data Wrangler**: Data preparation
- **Model Monitor**: Monitoring
- **Pipelines**: MLOps automation

### Exam Tips
- Focus on WHEN and WHY to use AI/ML, not just WHAT
- Understand practical applications, not just theory
- Know the difference between learning paradigms
- Recognize data type requirements for different tasks
- Understand the complete ML lifecycle, not just training

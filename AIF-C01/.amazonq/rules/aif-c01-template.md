# AWS AIF-C01 Exam Notes Template

When generating service/topic notes for AWS Certified AI Practitioner (AIF-C01), use this structure:

## 1. Service/Topic Overview
- **Purpose**: One-sentence description of what the service does
- **Service Level**: Global | Regional | AZ-Specific
- **Key Use Cases**: 3-5 primary scenarios
- **Exam Weight**: Estimated importance (High/Medium/Low)
- **Exam Domain**: Map to one of:
  - Domain 1: Fundamentals of AI and ML (20%)
  - Domain 2: Fundamentals of Generative AI (24%)
  - Domain 3: Applications of Foundation Models (28%)
  - Domain 4: Guidelines for Responsible AI (14%)
  - Domain 5: Security, Compliance, and Governance for AI Solutions (14%)
- **Task Statements**: Specific exam objectives this service addresses

## 2. Exam Keyword Mapping
- **Trigger Words**: Keywords that signal this service in exam questions
- **Scenario Indicators**: Phrases that point to this service as the answer
- **Anti-patterns**: When NOT to use this service (common distractors)

## 3. Core Concepts
- **Architectural Patterns**: How the service fits in solutions
- **Service Limits**: Quotas, constraints, and boundaries
- **Data Flow**: Input → Processing → Output
- **Integration Points**: How it connects with other AWS services

## 4. Key Features & Components
- **Exam-Tested Features**: Features frequently appearing in questions
- **Configuration Options**: Critical settings and parameters
- **API/SDK Highlights**: Key operations and methods
- **Integrations**: Services commonly paired with this one

## 5. Exam Focus Areas
- **Common Question Types**: 
  - Scenario-based selection questions
  - Feature comparison questions
  - Troubleshooting questions
  - Best practice questions
- **Gotchas**: Tricky aspects and common misconceptions
- **Comparison Points**: How this differs from similar services
- **Decision Trees**: When to choose this over alternatives

## 6. Best Practices
- **Security**: IAM, encryption, network isolation, data protection
- **Cost Optimization**: Pricing model, cost-saving strategies
- **Performance**: Optimization techniques, scaling considerations
- **Operations**: Monitoring, logging, maintenance, troubleshooting

## 7. Hands-On Labs
- **Quick Setup**: Minimal steps to get started
- **Practical Exercises**: 2-3 hands-on scenarios
- **CLI Commands**: Essential AWS CLI examples
- **Console Walkthrough**: Key console navigation steps

## 8. Sample Questions
Provide 2-3 AIF-C01 style questions:
- Multiple choice with 4 options
- Scenario-based context
- Include correct answer with detailed explanation
- Explain why wrong answers are incorrect

---

**Exam Tip**: Focus on understanding WHEN and WHY to use each service in real-world scenarios, not just memorizing features. The AIF-C01 exam tests practical decision-making for AI/ML solutions on AWS.
